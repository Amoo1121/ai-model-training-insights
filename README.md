# ai-model-training-insights

这是一个面向大模型训练资料整理的文档仓库，当前主要内容位于 `AI模型训练洞察/` 目录下。

## 内容概览

项目按模型与通用主题组织，便于横向比较与后续补充：

- `AI模型训练洞察/GPT/`
- `AI模型训练洞察/LLaMA/`
- `AI模型训练洞察/Claude/`
- `AI模型训练洞察/DeepSeek/`
- `AI模型训练洞察/GLM/`
- `AI模型训练洞察/QWEN/`
- `AI模型训练洞察/MiniMax/`
- `AI模型训练洞察/闭源模型/`
- `AI模型训练洞察/common/`

## 说明

- 各模型目录下的 `training.md` 记录对应模型的训练相关洞察。
- `common/` 目录用于整理通用的数据准备、训练技巧等主题。
- 更详细的中文说明请见：`AI模型训练洞察/README.md`
